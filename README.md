# GitHub Pages Prototype — Alexander Arthur Gordon

This package is a static-site prototype mirroring the current Framer website direction.

## Included
- `index.html` home page prototype
- `articles/` placeholder article pages
- `css/styles.css` global styling
- `js/script.js` small interactions and scroll atmosphere
- `assets/` image placeholders pulled from the current website work

## GitHub Pages setup
1. Create a new GitHub repository.
2. Upload all files in this package to the repository root.
3. In **Settings → Pages**, set source to **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Save. GitHub Pages will publish the site.

- The hero video is referenced as `assets/hero-background.mp4` but no video file is bundled; add your own.
